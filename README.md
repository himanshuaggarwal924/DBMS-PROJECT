# Project Overview :
The Smart Travel Planning and Recommendation System is a full-stack web application that helps users discover and organize travel information for different cities. Users can search for a city and explore hotels, restaurants, and tourist attractions, view ratings and reviews, and plan their trips efficiently.

The system uses a MySQL database to store and manage travel data, enabling efficient filtering, searching, and analytics. It also demonstrates practical implementation of Database Management System (DBMS) concepts such as ER modeling, relational schema design, SQL queries, joins, constraints, and aggregation.

This project is developed as part of the Database Management System Lab mini-project.
